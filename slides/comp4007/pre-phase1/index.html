1. Plantuml vs Argouml

Two uml tools are in the game

Argouml

Argouml is suggest by Joe in the specification, I downloaded it and it is a pure wysiwyg(what you see is what you get) diagram editor.

The software is very easy to use, however, it is based on digram, so team collaboration is not ideal

Plantuml

Plantuml is a very powerful uml tools based on pure text files.


@startuml
left to right direction
skinparam packageStyle rect
actor customer
actor clerk
rectangle checkout {
  customer -- (checkout)
  (checkout) .> (payment) : include
  (help) .> (checkout) : extends
  (checkout) -- clerk
}
@enduml
